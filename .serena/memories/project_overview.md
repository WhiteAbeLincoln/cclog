# cclog Project Overview

## Purpose
cclog is a Go command-line tool that parses Claude Code conversation logs (JSONL format) and converts them to human-readable Markdown. It features both a powerful interactive TUI mode and flexible CLI processing.

## Key Features
- **Interactive TUI Mode**: Rich terminal interface to browse, preview, and manage conversation logs
- **Live Markdown Preview**: Instant preview of JSONL files rendered as Markdown
- **Claude CLI Integration**: Resume conversations directly from the TUI
- **Flexible CLI Mode**: Process files or directories from command line
- **Message Filtering**: Intelligent filtering to remove system noise
- **Clean Output**: Beautiful Markdown formatting with proper timezone handling

## Core Value Proposition
Transforms Claude Code's raw conversation logs into readable, navigable documentation that can be easily browsed, searched, and resumed.
